# Memory Game

## Description du projet

Ce projet est un jeu de mémoire développé en HTML, CSS et JavaScript.  
Le joueur doit retourner des cartes et retrouver les paires identiques le plus rapidement possible.

Le jeu propose plusieurs niveaux de difficulté afin d’augmenter progressivement le nombre de cartes et la difficulté du jeu.

---

## Technologies utilisées

- HTML
- CSS
- JavaScript (Vanilla)

---

## Fonctionnalités principales

- Génération automatique des cartes
- Système de paires à trouver
- Trois niveaux de difficulté (facile, moyen, difficile)
- Chronomètre pour mesurer le temps
- Système de score
- Mélange aléatoire des cartes à chaque partie
- Détection de victoire

## Difficultés rencontrées

- Gestion de la logique des paires
- Synchronisation entre les cartes retournées
- Gestion du temps et du score
- Organisation du code JavaScript

---

## Solutions apportées

Pour résoudre ces problèmes, j’ai :
- testé plusieurs versions du code
- utilisé la console pour le débogage
- séparé les fonctions pour mieux organiser la logique
- vérifié la documentation JavaScript
- corrigé progressivement les erreurs de logique
